# HR Analytics Dashboard 📊
Developed by Abhay Kumar (B.Tech CSE)
📌 Project Overview

This project presents an HR Analytics Dashboard built using Microsoft Power BI to analyze employee data and understand workforce trends within an organization. The dashboard helps HR teams identify patterns related to employee attrition, demographics, job roles, and salary distribution.

The goal of this project is to convert raw HR data into interactive visual insights that support better decision-making and workforce planning.
## Dataset Used
<a href="https://github.com/dd0a426/HR-Analytics-Dashboard/blob/main/HR_Analytics.csv">Dataset</a>
  
# 🎯 Objectives

Analyze employee attrition and attrition rate.

Understand employee demographics such as age, gender, and education.

Identify job roles and salary slabs with higher attrition.

Evaluate employee experience and years at company.

# 📊 Dashboard Features

Total Employees Count

Total Attrition and Attrition Rate

Average Age, Salary, and Work Experience

Attrition by Education Level

Attrition by Age Group

Attrition by Salary Slab

Attrition by Job Role

Attrition Trend by Years at Company

Department Filter (HR, R&D, Sales)

Attrition by Gender

# 🛠 Tools & Technologies Used

Microsoft Power BI

Power Query – Data Cleaning & Transformation

DAX (Data Analysis Expressions) – Calculations & Measures

Data Visualization Techniques

# 📈 Key Insights

Certain job roles experience higher attrition compared to others.

Employees with lower salary slabs tend to leave more frequently.

Age group and education level also influence attrition patterns.

Department-based filtering helps identify attrition trends across departments.
